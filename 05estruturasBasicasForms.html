Formulários de entrada de dados e envio ao Servidor

<form> Tag de formulário </form>

Formulário de texto:

  • Rótulo tem que ser criado (label) dentro do formulário e necessáriamente precisa estar associado a um input e id.
  • Para iniciar com um valor padrão utilize o parâmetro "value".
  • Observe os parâmetros do input para que o formulário fique de uma forma mais amigável com a interface utilizando size, placeholder, password e outros...
  • O label está associado ao id do input.









Formulário de caixa de seleção:

Para um checkbox alguns pontos são importantes:

  • No input do tipo checkbox:
  • ID: é a identificação única do checkbox;
  • NAME: é o nome/chave do campo valor á ser enviado;
  • VALUE: é o valor que será enviado;
RESUMO: O conteúdo do VALUE do nome NAME será enviado ao servidor para processamento.
