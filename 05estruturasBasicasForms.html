Formulário de texto:

  • Rótulo tem que ser criado (label) dentro do formulário e necessáriamente precisa estar associado a um input e id.
  • Para iniciar com um valor padrão utilize o parâmetro "value".
  • Observe os parâmetros do input para que o formulário fique de uma forma mais amigável com a interface utilizando size, placeholder, password e outros...
